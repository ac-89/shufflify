About

Shufflify is a project to demostrate my usage of APIs.

The song titles are pulled from the quoteable API (https://github.com/lukePeavey/quotable)

Artist names are pulled from the MediaWiki API (https://www.mediawiki.org/wiki/API:Main_page)

Currently, the album art is not pulled from an API but I hope to change that soon.
